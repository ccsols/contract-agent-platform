UI STYLE — SWISS EDITORIAL
v0.1 · STATIC PREVIEW

条款不是噪音，是结构。

瑞士编辑风用“报纸级排版”组织信息：强网格、清晰层级、克制的装饰。适合强调专业与可信度的法务、金融、B2B SaaS，尤其是内容密度高但希望仍显冷静的产品。

设计要点

  • 超大标题 + 窄副标题第一眼给出产品主张
  • 细线与留白分区避免阴影堆叠造成“软塌”
  • 少量纯黑强调CTA 与关键导航保持锋利

01 栅格

12 列栅格让卡片宽度有节奏；移动端直接叠成单栏，阅读动线更直。

02 字体

系统无衬线即可；通过字重与字号阶梯制造层级，少用花体字。

03 颜色

主色通常接近黑白；若需要品牌色，建议只用于一条“编辑线”或小标签。

如果你希望界面像“精密仪器”而不是“氛围海报”，这一版通常最耐看、也最容易长期维护。
