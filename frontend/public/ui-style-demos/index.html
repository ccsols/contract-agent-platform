以下为互不依赖的纯静态 HTML，可直接双击打开，或在 Next 开发服务器下访问 /ui-style-demos/ 路径。

极光玻璃 · 同系变体

  • 深 · 经典
    极光玻璃（Aurora Glass）
    深色底 + 网格渐变、磨砂玻璃与柔和光晕，原版参考。
  • 浅 · 冷暖平衡
    蛋白石晨雾（Opal Dawn）
    亮底 + 桃粉 / 天蓝 / 薄荷光斑，亲和力强的浅色玻璃。
  • 浅 · 粉紫
    薰衣草牛奶（Lilac Milk）
    柔雾粉紫系，温和可信，适合品牌化程度较高的 B2B。
  • 浅 · 冷调
    极地雾蓝（Arctic Mist）
    冰蓝雾白，理性清爽，偏合规 / 数据 / 审计叙事。
  • 浅 · 暖调
    蜂蜜静谧（Honey Serene）
    蜂蜜燕麦暖浅底，协作与客户向界面更「有人味」。
  • 浅 · 中性
    珍珠暮灰（Pearl Dusk）
    银灰 + 极淡靛紫高光，最中性、最易后续换 accent 品牌化。

其他方向

  • 排版 · 理性
    瑞士编辑风（Swiss Editorial）
    大字号、强网格、高对比，偏 Linear / 杂志式信息层级。
  • 态度 · 记忆点
    新粗野主义（Neo-Brutalist）
    粗边框、硬阴影、高饱和色块，适合品牌年轻化与营销落地页。
  • 开发者 · 终端
    午夜工具台（Midnight DevTools）
    深色底、等宽点缀、细网格与状态色，适合技术型控制台。
  • 温和 · 效率
    柔感便当格（Soft Bento）
    大圆角、轻阴影、暖灰背景，偏 Apple / Notion 式日常工具感。
